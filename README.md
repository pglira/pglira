# Hi!

My name is Philipp Glira and I'm a scientist at the awesome [Austrian Institute of Technology](https://www.ait.ac.at) (AIT).

## Notable GitHub contributions

- [simpleICP](https://github.com/pglira/simpleICP) — Implementations of a simple Iterative Closest Point algorithm in C++, Python, Julia, Matlab, and Octave
- [Point_cloud_tools_for_Matlab](https://github.com/pglira/Point_cloud_tools_for_Matlab) — A collection of point cloud processing tools for Matlab
- [robot-mapping-exercises-by-stachniss](https://github.com/pglira/robot-mapping-exercises-by-stachniss) — Solutions to the exercises from Cyril Stachniss's robot mapping lecture
- [kalman-filter-example-by-welch-and-bishop](https://github.com/pglira/kalman-filter-example-by-welch-and-bishop) — C++ implementation of the example from Welch & Bishop's *Introduction to the Kalman Filter*
- [publications](https://github.com/pglira/publications) — An archive of some of my scientific publications
- [cloudcompare-sbf-io](https://github.com/pglira/cloudcompare-sbf-io) — Matlab support for CloudCompare's SBF point cloud format
- [archive-images](https://github.com/pglira/archive-images) — Archive images into year/month/day folders based on EXIF timestamps
- [download-to-year-month-day-dir-chrome-extension](https://github.com/pglira/download-to-year-month-day-dir-chrome-extension) — Chrome extension that sorts downloads into year/month/day folders
- [do-re-mi-fa-so-la-si-done](https://github.com/pglira/do-re-mi-fa-so-la-si-done) — A minimal Python + Qt desktop todo app
- [psst](https://github.com/pglira/psst) — Voice-to-text for Linux powered by whisper.cpp
- [vimiv-qt](https://github.com/pglira/vimiv-qt) — My own fork of the awesome vimiv-qt image viewer
